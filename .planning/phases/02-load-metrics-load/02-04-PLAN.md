# 02-04: Metrics reconciliation ‚Äî P10 mitigation

## Goal

Compare client-side `RequestTelemetry` against vLLM's `/metrics` endpoint.
`ReconciliationReport.delta_pct` must be `‚â§ 2.0` for the metrics that drive
goodput (success count, prompt tokens, completion tokens, mean TTFT). If
exceeded, `gate_passes() = False` ‚Äî the orchestrator refuses to record a
benchmark run as evidence (P10).

## Why this is its own plan

Without ground-truth reconciliation the benchmark is a mirror of itself.
vLLM exposes counters (`vllm:prompt_tokens_total`, `vllm:generation_tokens_total`,
`vllm:e2e_request_latency_seconds_sum`) on `/metrics`. We parse the
Prometheus text format (no external deps ‚Äî hand-rolled parser using
`prometheus_client.parser` if available, fallback to regex). Then compare
client totals to server totals.

## Files

| File | Purpose |
|------|---------|
| `core/metrics.py` | `parse_prometheus(text) ‚Üí dict[str, float]` + `reconcile(...)` |
| `core/reconcile.py` | `ReconciliationReport` Pydantic model |
| `tests/test_reconcile.py` | Synthetic vLLM `/metrics` text fixtures + deltas |

## Reconcile contract

```python
class ReconciliationReport(BaseModel):
    model_config = ConfigDict(extra="forbid")
    success_count_delta_pct: float
    prompt_tokens_delta_pct: float
    completion_tokens_delta_pct: float
    mean_ttft_delta_pct: float
    window_s: float
    n_client_requests: int
    n_server_requests: int

    def gate_passes(self, threshold: float = 2.0) -> bool: ...

def reconcile(
    client: list[RequestTelemetry],
    vllm_metrics_text: str,
    window_s: float = 60.0,
) -> ReconciliationReport: ...
```

Parses vLLM's `vllm:e2e_request_latency_seconds_sum`, `vllm:request_success_total`,
`vllm:prompt_tokens_total`, `vllm:generation_tokens_total` from Prometheus
text. Computes deltas vs client-side aggregates. Tests construct fake
`vllm_metrics_text` strings with known values.

## Test strategy

- `test_parses_prometheus_text_basic` ‚Äî known metric line ‚Üí float
- `test_reconcile_zero_delta_when_equal` ‚Äî synthetic match ‚Üí 0.0 delta
- `test_reconcile_detects_drift` ‚Äî server tokens 2x client ‚Üí delta_pct = 100.0
- `test_reconcile_gate_passes_under_threshold` ‚Äî 1.5% delta ‚Üí True
- `test_reconcile_gate_fails_over_threshold` ‚Äî 3% delta ‚Üí False
- `test_reconcile_handles_missing_metrics` ‚Äî empty body ‚Üí 0 server, large delta
