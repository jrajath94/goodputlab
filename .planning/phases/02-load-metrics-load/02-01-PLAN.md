# 02-01: Trace data model + open-loop arrival processes

## Goal

Establish the foundational types and arrival-process primitives that all
load generators, replay engine, and metrics reconciler will compose. After
this plan, the project has a single canonical representation of:

1. A **trace** (list of request specs + arrival config + seed)
2. A **per-request telemetry** record (LOAD-05 schema)
3. **Open-loop arrival processes** ŌĆö Poisson + ON/OFF (LOAD-04)
4. A **deterministic sampler** for byte-identical replay (LOAD-07)

## Why first

Every subsequent Phase 2 plan consumes these types. Without a single
canonical `RequestTelemetry` schema, the metrics reconciler (plan 02-04)
cannot compare its in-memory log against vLLM's `/metrics` output.
Without open-loop arrivals, the chat/RAG/agentic generators cannot emit
realistic bursts (LOAD-04). Without deterministic sampling, replay
(LOAD-07) is not byte-identical.

## Schema (canonical, in `core/trace.py`)

```python
class WorkloadType(str, Enum):
    CHAT = "chat"
    RAG = "rag"
    AGENTIC = "agentic"

class SloClass(str, Enum):
    INTERACTIVE = "interactive"   # TTFT p95 < 800ms
    BATCH = "batch"               # throughput-first, no TTFT guarantee

class RequestSpec(BaseModel):
    request_id: str               # uuid4 (deterministic from seed, not uuid4())
    slo_class: SloClass
    workload: WorkloadType
    prompt_tokens: int            # input length target (post-tokenization, generator-side)
    output_tokens: int            # output length target
    prompt_text: str              # actual prompt (chat: user msg; RAG: system+context+query; agentic: tool result)

class RequestTelemetry(BaseModel):
    request_id: str
    enqueue_ts_ns: int            # client clock (perf_counter_ns)
    ttft_ms: float | None         # None if request failed before first token
    per_token_ts_ns: list[int]    # monotonic client timestamps per emitted token
    completion_ts_ns: int | None
    status_code: int              # HTTP status from upstream
    error: str | None             # error message if status >= 400 or transport failure
    routed_pool: str | None       # "prefill" | "decode" | "colocated" ŌĆö set by router (Phase 3+)

class ArrivalConfig(BaseModel):
    process: Literal["poisson", "on_off"]
    rate_per_sec: float           # for poisson: lambda; for on_off: on-phase arrival rate
    seed: int
    on_duration_s: float | None   # ON/OFF only
    off_duration_s: float | None  # ON/OFF only

class Trace(BaseModel):
    workload: WorkloadType
    seed: int
    duration_s: float
    arrival: ArrivalConfig
    requests: list[RequestSpec]   # order is irrelevant; arrival times are computed from config
    metadata: dict[str, str]      # free-form: model, hardware, version, etc.
```

## Arrival processes (in `loadgen/arrival.py`)

```python
class PoissonArrival:
    """Exponential inter-arrival; deterministic from (seed, n)."""
    def __init__(self, rate_per_sec: float, seed: int) -> None: ...
    def sample(self, n: int) -> list[float]:
        """Return n arrival offsets in seconds from t=0."""

class OnOffArrival:
    """Two-state Markov: ON emits at rate, OFF emits nothing."""
    def __init__(self, on_rate: float, on_dur: float, off_dur: float, seed: int) -> None: ...
    def sample(self, duration_s: float) -> list[float]:
        """Return arrival offsets in seconds, sampled up to duration_s."""

class OpenLoopScheduler:
    """Yields (RequestSpec, arrival_ts_ns) pairs in time order; never blocks."""
    def __init__(self, trace: Trace, clock: Callable[[], int] = time.perf_counter_ns) -> None: ...
    def __iter__(self) -> Iterator[tuple[RequestSpec, int]]: ...
```

### Determinism contract (LOAD-07)

`PoissonArrival(seed=42).sample(1000)` and `OnOffArrival(seed=42, ...).sample(60.0)`
must produce **bit-identical** results across two runs on the same Python
version. We use `random.Random(seed)` (Mersenne Twister) for stability
across cPython 3.11+ ŌĆö NOT `numpy.random.default_rng()` (PCG64, bit
changes between numpy versions).

Byte-identical replay test asserts:
```python
a = PoissonArrival(rate=10, seed=42).sample(1000)
b = PoissonArrival(rate=10, seed=42).sample(1000)
assert a == b  # list equality on float == exact bit equality
```
