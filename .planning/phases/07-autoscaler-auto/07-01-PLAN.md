# 07-01: PID P:D autoscaler

## Goal

A `PidController` + `PoolAutoscaler` that adjusts the P:D pool ratio
based on observed queue depths.  The controller is the discrete-time
PID form:

    u[t] = Kp·e[t] + Ki·(I[t-1] + e[t]·dt) + Kd·(e[t] - e[t-1]) / dt

where ``e[t] = queue_depth[t] - target_depth``.

The autoscaler emits integer ``+N`` / ``-N`` deltas per tick (positive
= scale up, negative = scale down).  Scale-down respects a **drain
protocol** (AUTO-02): wait until in-flight requests on the affected
pool hit 0 before actually removing the replica.

## PID contract

```python
class PidGains(BaseModel):
    kp: float
    ki: float = 0.0
    kd: float = 0.0

class PidController:
    def __init__(self, gains: PidGains, output_min: float, output_max: float): ...
    def reset(self) -> None: ...
    def update(self, error: float, dt: float) -> float: ...
```

Anti-windup: when output saturates at `output_min` / `output_max`,
the integrator is held (not accumulated).  Otherwise a long-running
positive error with low target would saturate and never recover.

## Autoscaler contract

```python
class PoolTopology(BaseModel):
    pool: Pool
    replicas: int = Field(ge=0)
    target_queue_depth: int = 16

class AutoscalerDecision(BaseModel):
    pool: Pool
    delta: int           # signed; +1 scale up, -1 scale down, 0 idle
    reason: str          # "queue_high" | "queue_low" | "drain_wait" | ...
    drained: bool        # True if this tick was a scale-down that completed

class PoolAutoscaler:
    def __init__(
        self,
        controllers: dict[Pool, PidController],
        min_replicas: int = 1,
        max_replicas: int = 8,
        step_size: int = 1,
    ): ...
    def tick(
        self,
        topology: dict[Pool, PoolTopology],
        queue_depths: dict[Pool, int],
        in_flight: dict[Pool, int],
        dt: float = 1.0,
    ) -> list[AutoscalerDecision]: ...
```
